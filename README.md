📡 WiFi Logger Project
📌 Project Description

The WiFi Logger Project is a software-based application that scans and records nearby WiFi network details such as SSID, signal strength, and timestamps. The system stores logs for analysis and uses a Telegram Bot to send real-time updates and allow remote access through commands.

🎯 Objectives

Monitor available WiFi networks

Log network information automatically

Provide remote access using Telegram Bot

Demonstrate automation using programming

🛠 Technologies Used

Python

Git & GitHub

Telegram Bot API

📂 Project Structure

src/ – Source code files

logs/ – Generated WiFi logs

docs/ – Report and presentation

screenshots/ – Output images

⚙️ Installation
git clone https://github.com/username/wifi-logger.git
cd wifi-logger
pip install -r requirements.txt
▶️ How to Run
python main.py
🤖 Telegram Commands

/start – Start bot

/logs – Get latest WiFi logs

/status – Check system status

📊 Output

WiFi data is stored in log files and can be accessed via Telegram.
